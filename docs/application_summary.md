# OpenClaw 参赛项目公开摘要

## 项目名称

南大数智安全官 NJU Guardian——基于 OpenClaw 的校园电信网络诈骗智能识别与防护 Agent

## 项目简介

电信网络诈骗在大学生群体中高发，尤其集中在冒充辅导员、兼职刷单、校园贷、二手交易、钓鱼链接、AI 换脸拟声等场景。通用反诈产品能覆盖大类风险，但对校园语境中的具体话术、身份关系和处置路径支持不足。

NJU Guardian 是一个面向校园场景的反诈 AI Agent 原型。用户可以粘贴可疑短信、聊天记录或链接，系统返回风险等级、命中案例、识别依据、三步行动建议和紧急联系渠道。当前本地 Demo 使用 Gradio 实现，检测引擎采用「关键词 / 正则规则」与「BGE-zh 向量召回」双路融合。

## 当前交付

- 本地 Gradio Demo：文本检测、截图 OCR、链接核验
- 50 类结构化诈骗案例知识库
- 双路召回检测引擎：规则路 + 向量路
- 可转发反诈卡片与 HTML 报告下载
- CI 冒烟测试、贡献指南、双许可证
- 架构图、对话卡片、小程序首页 mockup

## OpenClaw 决赛方向

后续计划在 OpenClaw 平台上重建多 Agent 流程：

- 主控 Agent：识别输入类型并进行任务路由
- 文本风险 Agent：RAG 检索、规则匹配、风险评级
- 多模态截图 Agent：OCR、二维码解析、转账界面识别
- 链接电话 Agent：URL 风险接口、短链还原、号码风险提示
- 案例回流 Workflow：用户授权后自动脱敏并进入知识库

## 数据与隐私原则

- 原型阶段默认不持久化用户输入
- 公开知识库只收录可核验公开来源
- 不收录微信群截图、内部材料、真实受害者隐私信息
- 后续真实校园案例必须先经脱敏 Workflow 处理
