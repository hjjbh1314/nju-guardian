# 知识库来源索引

`demo/knowledge_base.json` 的每条案例都带有 `source` 字段。该字段采用短标形式，指向公开发布的反诈材料、主流媒体报道、高校保卫处公开提示或监管机构公告。

本页用于解释短标含义，方便后续贡献者补充可访问链接、核验出处、替换失效来源。若新增案例，请优先使用可公开访问、可长期检索的来源。

## 使用原则

1. `source` 字段必须能解释案例来自哪类公开材料，不能写 `TBD`、`内部材料`、`微信群截图` 等不可核验来源。
2. 如果一个案例来自多类材料，使用 `+` 分隔，例如 `国家反诈手册2023·KB1 + 川农2025·案例1`。
3. 后续新增案例建议在 PR 描述中附上具体 URL；若 URL 容易失效，也应保留机构名、标题、发布时间等可检索信息。
4. 不收录未公开材料、匿名自述、真实聊天截图或可能暴露受害者身份的信息。
