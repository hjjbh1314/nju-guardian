`demo/knowledge_base.json` 中的 50 类诈骗案例，以及 `visuals/` 目录下的全部 HTML 视觉素材，按 [Creative Commons Attribution 4.0 International](https://creativecommons.org/licenses/by/4.0/) 协议发布。

## 你可以

- **共享** —— 在任何媒介以任何形式复制、发行
- **演绎** —— 修改、转换或以本作品为基础进行创作
- 用于任何目的，甚至商业目的

## 你需要

**署名（Attribution）** —— 在显著位置标注：

```
Cases adapted from "南大数智安全官 NJU Guardian" knowledge base v0.2.0
(https://github.com/hjjbh1314/nju-guardian) under CC BY 4.0.
Original sources: see each case's `source` field.
```

## 重要说明

每条 case 的 `source` 字段指向的是**该案例最初的公开来源**（如国家反诈中心、央视、公安部刑侦局等）。这些原始材料的版权归各自的原作者所有，本仓库只是在公开来源基础上做了**结构化整理与改写**。

你转用本知识库时：

1. **必须**在你的产品/论文/演示里标注本仓库 + 案例的原始来源
2. **不得**声称这些案例为你独立创作
3. **建议**遵循原始来源的引用规范（部分政府机构对引用有额外要求）
