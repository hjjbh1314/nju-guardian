# 对话式反诈 Agent · 真实运行实录

> 以下是 `agent/guardian_agent.py` 用 **DeepSeek（deepseek-chat）** 实际跑出来的对话（2026-06-06），**未经修改**。
> 它体现的是"Agent"而非"分类器"：**会追问、会调用案例库工具查证、最后给可执行处置**。
> 自己复现：`pip install openai` → `export DEEPSEEK_API_KEY=...` → `python agent/guardian_agent.py`。
> 场景取自"南开学生闲鱼买相机被诱导共享屏幕、分22次转账被骗220万"（2026-06-05）的同类话术。

---

**你>** 我在闲鱼买个相机，客服说我操作错了被多扣了钱，要帮我退

**安全官>** 同学你好，听起来确实让人有点紧张。我先帮你理一理情况——

你说的"客服"是在闲鱼平台内联系你的，还是通过其他方式（比如短信、QQ、微信）联系你的？另外，他有没有让你下载什么App、共享屏幕，或者让你先转账才能退款？

---

**你>** 他让我下了个会议软件开了屏幕共享，说要核对流水，我还没转钱但他在指导我操作

**安全官>** 　〔查案例库：闲鱼 客服 屏幕共享 退款 多扣钱 会议… → 命中 KB-004, KB-029〕

查到了，你遇到的正是典型的 **冒充电商客服 + 屏幕共享诈骗**，风险等级：**高**。

### 为什么这是诈骗？

1. **闲鱼根本没有"客服"会主动联系你退款**——正规平台退款只在 App 内操作，不会让你下载会议软件。
2. **屏幕共享 = 把你的手机屏幕完全暴露给对方**，你输入的密码、收到的验证码，对方都能实时看到。一旦你输入银行卡信息或验证码，钱就会被转走。
3. 国家反诈预警要点第四条明确：**自称客服主动退款理赔，要求提供银行卡和验证码的，都是诈骗。**

### 你现在立刻做这几件事：

1. **马上关闭屏幕共享**，退出那个会议软件，然后卸载它。
2. **立即拨打 96110**（全国反诈专线），说明情况，让警方帮你核查账户安全。
3. **给银行客服打电话挂失或冻结银行卡**（手机银行一般有一键挂失功能），防止信息已被窃取。
4. **修改闲鱼密码和支付密码**，不要和之前用过的密码重复。
5. 保留所有聊天记录、截图，必要时联系 **南大保卫处 81686110** 求助。

**你还没转账，这是万幸！** 但屏幕共享期间对方可能已经看到了你的个人信息，所以上面几步一定要尽快做。有任何后续情况随时找我！

*Agent 全程：先追问关键细节（联系渠道 / 是否下载App、共享屏幕、转账）→ 调用 search_cases 查证案例库 → 依据案例库 + 国家反诈预警要点下研判 → 给出按优先级排序的可执行处置，并强调"还没转账"的止损窗口。这是一次真实的 function-calling agentic 多轮对话。*
